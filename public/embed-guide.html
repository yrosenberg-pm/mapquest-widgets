MapQuest Widgets — Embedding Guide

Add interactive map widgets to your website in under 5 minutes.

Quick Start

1
Get your API key
Sign up or log in at developer.mapquest.com and copy your Consumer Key from the dashboard.
2
Choose a widget
Pick a widget from the table below and note its URL path (e.g. /directions).
3
Add the iframe to your page
Paste the snippet below into your HTML, replacing YOUR_MAPQUEST_API_KEY with your key and directions with your chosen widget path. 
<iframe
  src="https://widgets.mapquest.com/directions?apiKey=YOUR_MAPQUEST_API_KEY&darkMode=0&showBranding=1"
  width="100%"
  height="640"
  style="border:0; border-radius:12px; overflow:hidden"
  loading="lazy"
  allow="geolocation"
></iframe>
That's it! The widget will load with your API key. All MapQuest API calls (map tiles, geocoding, directions, search) are billed to the key you provide.

Customization Options

Add any of these URL parameters to control the widget appearance:

ParameterValuesDescription
apiKeyYour keyMapQuest API key (required)
darkMode0 or 1Light or dark theme
accentColorHex color, e.g. #2563ebPrimary brand color
fontFamilyCSS font stackCustom font family
borderRadiusCSS value, e.g. 0.5remCorner rounding
showBranding0 or 1Show "Powered by MapQuest" footer
companyNameTextYour company name (co-branding)
companyLogoImage URLYour logo URL (co-branding)

Example with customization

<iframe
  src="https://widgets.mapquest.com/directions?apiKey=abc123&darkMode=1&accentColor=%23e11d48&companyName=Acme%20Logistics&showBranding=1"
  width="100%"
  height="640"
  style="border:0; border-radius:12px"
  loading="lazy"
  allow="geolocation"
></iframe>

FAQ

How is usage billed?

All MapQuest API calls (map tiles, geocoding, directions, search) are made using the apiKey you provide. Usage is billed to your MapQuest developer account.

Can I use my own domain?

Yes. The iframe src points to the hosted widget service. Your users see your site with the widget embedded — no server setup needed on your end.

How do I test before going live?

Use the Embed Tester page to preview any widget with your API key before adding it to your site.

Can I style the widget to match my brand?

Yes — use the URL parameters above (accentColor, fontFamily, darkMode, companyName, companyLogo) to match your brand. You can also use the visual Customize panel in the main app to generate the embed code with your settings pre-applied.
